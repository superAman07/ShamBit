# Security Policy
# 
# We take security seriously at ShamBit. If you discover a security 
# vulnerability, please report it to us responsibly.
#
# Please do NOT:
# - Publicly disclose the vulnerability before we've had a chance to fix it
# - Access or modify data that doesn't belong to you
# - Perform any attacks that could harm our users or services
#
# Please DO:
# - Give us reasonable time to investigate and fix the issue
# - Provide detailed information about the vulnerability
# - Use the contact information provided above
#
# We appreciate your help in keeping ShamBit secure!
