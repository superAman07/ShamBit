## ğŸ¯ Quick Action Reference

### Admin Quick Actions by Scenario

#### Scenario: Customer Not Home
```
1. Click "Record Delivery Attempt"
2. Select "Customer Not Available"
3. Add notes: "Called customer, no answer"
4. Click "Contact Customer"
5. Select "Phone"
6. Add notes: "Left voicemail to reschedule"
7. Click "Retry Delivery"
8. Set new delivery time
9. Done! Order back in delivery queue
```

#### Scenario: Payment Verification Needed
```
1. Click "Put on Hold"
2. Select "Payment Verification Required"
3. Add notes: "Checking with payment gateway"
4. Verify payment
5. Click "Release Hold & Continue"
6. Done! Order resumes normal flow
```

#### Scenario: Customer Wants Return
```
1. Customer requests return (within 7 days)
2. Review return request
3. Click "Approve Return"
4. Check "Restock items"
5. Add notes: "Quality issue confirmed"
6. Schedule pickup (separate action)
7. After return received, click "Initiate Refund"
8. Choose full/partial refund
9. Process refund in payment gateway
10. Click "Mark Refund Complete"
11. Enter transaction reference
12. Done! Customer refunded
```

#### Scenario: Wrong Address
```
1. Click "Record Delivery Attempt"
2. Select "Address Not Found"
3. Add notes: "Building number incorrect"
4. Click "Contact Customer"
5. Select "Phone"
6. Add notes: "Got correct address"
7. Update delivery address (if feature available)
8. Click "Retry Delivery"
9. Done! Delivery rescheduled
```
